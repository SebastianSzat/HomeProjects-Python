# HomeProjects-Python

A collection of small Python utility scripts built for personal and home use.

## Projects

| Folder | Description |
|--------|-------------|
| [MP3_Metadata_Cleaner](./MP3_Metadata_Cleaner/) | Batch tool to selectively clear ID3 metadata tags from MP3 files |
| [MP3_metadata_from_path](./MP3_metadata_from_path/) | Bulk-tag MP3 files with album, artist, title and track number derived from folder structure and filenames |
| [MP3_Online_Tagger](./MP3_Online_Tagger/) | Identify MP3 files via audio fingerprinting (AcoustID/fpcalc) and MusicBrainz online lookup, and fill in their ID3 tags automatically |
